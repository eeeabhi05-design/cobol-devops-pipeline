# COBOL CI/CD Pipeline

## Overview
End-to-end CI/CD pipeline for
COBOL program deployment using
modern DevOps tooling, built as
part of Mainframe Modernization
upskilling by a Senior Mainframe
Engineering Lead with 16 years
BFSI domain expertise.

## Technologies Used
- Jenkins (CI/CD automation)
- GitHub (source control)
- Git Bash / VS Code (development)
- IBM Z Explorer (z/OS connectivity)
- Zowe CLI (remote z/OS operations)
- COBOL / IBM Enterprise COBOL

## Pipeline Flow
1. COBOL source committed to GitHub
2. Jenkins detects push automatically
3. Compiles COBOL program
4. Deploys to TEST environment
5. Manual approval gate (production
   governance equivalent)
6. Deploys to PROD environment
